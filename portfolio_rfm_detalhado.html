📊 Segmentação de Clientes com RFM e KMeans

🔍 Visão Geral

Este projeto foi desenvolvido com o objetivo de segmentar clientes com base em seus comportamentos de compra, utilizando a metodologia RFM (Recência, Frequência, Monetário) combinada com o algoritmo de clusterização KMeans. Os resultados são apresentados em um painel interativo feito com Streamlit, permitindo análises visuais e tomadas de decisão orientadas por dados.

🎯 Objetivos

  • Aplicar técnicas de análise comportamental para categorizar clientes em grupos homogêneos.
  • Utilizar algoritmos de Machine Learning para identificar padrões relevantes.
  • Oferecer uma interface interativa e fácil de usar para stakeholders e equipes comerciais.

🧠 Tecnologias Utilizadas

  • Python 3.8
  • Pandas, Scikit-learn para análise e modelagem
  • Matplotlib e Seaborn para visualização
  • Streamlit para dashboard interativo
  • Docker e Docker Compose para conteinerização
  • GitHub Actions para CI/CD

⚙️ Pipeline de Processamento

  1. Leitura e preparação da base de dados de compras simulada
  2. Cálculo dos scores RFM por cliente
  3. Normalização dos dados com StandardScaler
  4. Aplicação do algoritmo KMeans para definir os clusters
  5. Exportação dos resultados com segmentação para um novo CSV
  6. Visualização e análise interativa via Streamlit

📊 Resultados Obtidos

  • Clusters bem definidos com insights sobre comportamento de compra
  • Segmentos identificados: VIP, leais, potenciais churn, novos, inativos
  • Painel interativo acessível via navegador

🧪 Validação e Métricas

Foi utilizado o método do cotovelo (elbow method) para definir a quantidade ideal de clusters. Análises estatísticas básicas foram feitas para interpretar a média de R, F e M em cada grupo.

📦 Execução com Docker

docker-compose build
docker-compose up

Acesse em: http://localhost:8501

Esse processo elimina dependências locais e garante que qualquer máquina com Docker rode o projeto com um único comando.
